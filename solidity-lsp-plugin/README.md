# solidity-lsp

Solidity language server for Claude Code, providing code intelligence and diagnostics. Powered by [juanfranblanco's Solidity Language Server](https://github.com/juanfranblanco/vscode-solidity).

## Supported Extensions

`.sol`

## Installing the LSP

### Via npm

```
npm install --global vscode-solidity-server
```

### Via bun

```
bun install --global vscode-solidity-server
```
